\section{Introduction}
This document presents comparison grids (2x2 layout) of Byzantine robustness results for CNN and SNN architectures under different data encoding schemes, extended up to $f=8$ Byzantine clients. The configurations compared are:
\begin{enumerate}
    \item \textbf{CNN}: Convolutional Neural Network on MNIST (`cnn\_complete\_plots\_extended\_f8`)
    \item \textbf{SNN Rate}: Spiking Neural Network with Rate encoding on MNIST (`snn\_complete\_plots\_rate\_extended\_f8`)
    \item \textbf{SNN Direct}: Spiking Neural Network with Direct encoding on MNIST (`snn\_complete\_plots\_direct\_extended\_f8`)
    \item \textbf{SNN NMNIST}: Spiking Neural Network on neuromorphic NMNIST (`snn\_complete\_plots\_nmnist\_extended\_f8`)
\end{enumerate}

Each grid illustrates the test accuracy under varying numbers of Byzantine clients ($f \in [0, 2, 4, 6, 8]$) and data heterogeneity levels ($\gamma \in [0.0, 0.33, 0.66, 1.0]$).
